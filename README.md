# Plumbler-PuzzleGame
A puzzle game similar to plumber .

# Build from C source code developed using C and GTK GUI library.

on Linux Ubuntu:

```bash
sudo apt install build-essential
sudo apt install libgtk-3-dev
cd ./c
sh ./Makefile
```
